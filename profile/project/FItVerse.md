# 🌌 FItVerse (Fit + Universe)
> 사용자의 신체에 맞춘 옷(Fit) 과 가상 세계(Universe) 의 융합

## 🎯 프로젝트 목표
- 온라인 의류 이커머스 시장의 한계인 “구매 전 직접 착용 불가” 문제를 해결하기 위해
- 이미지 분류 및 합성 기술을 활용한 가상 피팅(Virtual Try-On) 서비스를 개발
- 앱 및 웹 플랫폼을 통해 사용자에게 제공
